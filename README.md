## keys
| Key        | Action                     |
| ---------- | -------------------------- |
| Ctrl + Q   | Quit                       |
| Ctrl + E   | Maximise/unmaximise window |
| Ctrl + S   | Save file                  |
| Ctrl + D   | Save as                    |
| Ctrl + O   | Open                       |
